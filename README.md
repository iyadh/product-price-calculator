# Product Price Calculator

Demo on [Vercel](https://product-price-calculator.vercel.app/)

## Technical Stack

- Vite
- Vue 3
- Pinia
- TypeScript
- Tailwind CSS

## Project Setup

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
